# 💰 ExpenseTracker - Akıllı Bütçe ve Harcama Yönetimi

Bu uygulama, kullanıcıların günlük harcamalarını kaydedebildiği, kategorize edebildiği ve en önemlisi **canlı döviz kurları** üzerinden bütçelerini yönetebildiği modern bir Android uygulamasıdır.

## 🚀 Öne Çıkan Özellikler
* **Canlı Döviz Entegrasyonu:** Ücretsiz bir API üzerinden anlık döviz kurlarını çekerek harcamaları farklı para birimlerinde hesaplar.
* **Yerel Veritabanı (Room DB):** Harcamalarınız telefonunuzda güvenle saklanır, internet olmasa bile verilerinize ulaşabilirsiniz.
* **Kategori Bazlı Takip:** Harcamaları türlerine göre ayırarak bütçe analizi yapma imkanı.
* **Modern UI:** Jetpack Compose ile akıcı, hızlı ve şık bir kullanıcı deneyimi.
* **Asenkron İşlemler:** Coroutines ve Flow kullanarak kasmayan, hızlı bir yapı.

## 🛠️ Teknik Altyapı
* **Kotlin** - Projenin ana dili.
* **Jetpack Compose** - Modern arayüz tasarımı.
* **Room Database** - Yerel veri saklama (Persistence).
* **Retrofit** - Canlı döviz kurlarını çekmek için ağ yönetimi.
* **MVVM Architecture** - Temiz kod ve sürdürülebilir mimari.
* **Hilt (Dependency Injection)** - Bağımlılıkların profesyonelce yönetilmesi.

## 📦 Nasıl Çalıştırılır?
1. Projeyi Android Studio'da açın.
2. Emulator veya gerçek bir cihazda çalıştırın.
3. Uygulama açıldığında kurlar otomatik olarak güncellenecektir.
4. Harcamalarınızı ekleyip bütçenizi yönetmeye başlayın!
